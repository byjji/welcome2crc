### 투표 개편 — 선택지 ＋/－ · 마감일시 · 투표자 보기 · 참여율
- **선택지 입력을 줄글(textarea) → 행(row) 방식**으로: `＋ 선택지 추가` / 각 행의 `−` 로 삭제 (최소 2개 유지).
  투표 만들기·수정 모달이 같은 방식 (수정 모달은 기존 선택지가 행으로 채워짐)
- **투표 마감일시 지정**(선택): 날짜 + 시각. 시각을 비우면 그날 23:59 까지.
  `polls/{id}.deadline` 에 `"2026-07-20T18:00"` 로 저장하고, **지나면 운영진이 마감하지 않아도 자동 마감**
  (카드에 `7/20 18:00까지` → `7/20 18:00 마감됨`, 선택지 비활성). 화면을 켜둔 채 마감 시각이 지나도
  가장 가까운 마감 시각에 맞춘 타이머가 다시 그려줌
- **선택지별 인원수 버튼** — `참석 | 3명 60%` 처럼 표시되고, **인원수를 누르면 투표자 명단 모달**
  (이름 칩, 실시간 갱신 / 0명이면 비활성). 선택지 = [투표 버튼] + [인원수 버튼] 두 버튼 구조로 변경
- **참여율**: 카드 하단에 `4/25 참여` — 분모는 활동 크루원(member·admin) 수. 홈의 '진행 중인 투표'도 같은 표기
- 표 집계·크루원 수·마감 상태가 바뀌면 소식과 홈이 함께 갱신되도록 구독 연결 보강

### 공식 기록 — '기록(시간)' → '입상 순위'
- 페이지 관리의 기록 입력 칸을 **시간 형식 검증이 있는 필드 → 자유 텍스트('입상 순위')**로 변경
  (`1위`·`우승` 등 자유 입력, `results[].rank` 로 저장)
- 소개 페이지 표를 **`이름 | 입상 순위` 2열**로 정리(자동 순번 열 제거). 숫자로 시작하는 순위가 앞으로 정렬,
  숫자가 없는 값은 뒤로. 예전에 저장한 완주 기록(`time`)이 있으면 그대로 표시
- 종목(거리) 순서를 **짧은 거리부터**로 통일: `3km → 5km → 10km → 하프 → 풀코스`
  (등록 폼의 종목 선택, 관리 화면의 기록 묶음, 소개 페이지의 종목 탭 모두 동일)

### PWA 자동 업데이트 알림
- 앱을 켤 때·다시 화면에 돌아올 때마다 백그라운드로 `sw.js` 변경을 확인
- 새 버전이 감지되면 화면 하단에 **"새 버전이 출시되었습니다!"** 토스트 표시(업데이트·닫기 버튼)

- 첫 방문의 `clients.claim()` 으로 인한 불필요한 새로고침은 막음(사용자가 업데이트를 누른 경우에만 새로고침)
- 배포 방법: `sw.js` 의 **VERSION 을 올린 뒤 배포**하면 알림이 뜸 (안 올려도 온라인이면 최신 파일은 네트워크 우선으로 자동 반영되지만, '켜둔 앱'에 알림을 주려면 올릴 것)
- 확인: 변경 감지→토스트→대기→업데이트 클릭→새로고침 전체 사이클 8건 통과

- 세 페이지 head에 매니페스트·테마색·apple 메타 추가, 각 페이지에서 서비스 워커 등록(`js/lib/pwa.js`)

- 신청 정보 확장: 이름·성별(한 줄), 연락처(하이픈 자동), 사는 곳, 연령대, 대회 경력, 하고 싶은 말 — 성별/연령대/경력은 라디오 버튼
- **계정 아이디화**: 이메일 형식 대신 순수 아이디 사용 (영문·숫자·`.` `_` `-`), 대소문자 구분,
  내부적으로만 `@crc.ulsan` 가짜 도메인으로 변환 (대문자는 `+소문자` 인코딩으로 보존)
- 중복 계정이면 "이미 사용 중인 계정이에요" 안내 후 재입력
- **비밀번호 찾기 개편**: 가입 시 힌트 질문/답변 등록 (답변 대소문자 구분) →
  '비밀번호를 잊었나요'에서 계정+질문+답변이 맞으면 새 비밀번호 즉시 설정.
  비밀번호는 답변으로 암호화되어 `pwhints` 컬렉션에 저장 (PBKDF2 + AES-GCM)
- 내 정보 수정에서 힌트 등록/변경 가능, 비밀번호 변경 시 힌트 자동 갱신

### 소개 페이지 섹션 자동 관리 (커밋 193d323)
- '크루 가입 신청' 섹션(신청서 폼) 삭제 — 신청은 크루 공간에서만
- 내용 없는 섹션 자동 숨김: 크루 현황 숫자·핵심 가치·정기런 일정·공식 기록·갤러리·가입 안내·가입 문의
- **가입 안내(JOIN US)** 와 **가입 문의** 내용을 페이지 관리(운영진)에서 편집 — 저장하면 표시, 비우면 숨김

### 공지 목록 UI 개선 (커밋 193d323)
- 공지를 `고정핀 | 제목 | 날짜` 한 줄 목록으로, 누르면 펼쳐서 내용 확인
- 고정 글에만 📌 표시, 버튼 순서 고정(해제) → 수정 → 삭제
- 크루 공간 헤더에 '크루공간' 문구 복원, 로그인 후 이름 밑줄 제거
- 프로젝트 문서를 PROJECT.md 로 정리
